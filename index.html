Backbone Debugger

Chrome Developer Tools extension for debugging Backbone.js applications.

Install » Get the source »

Displays in real-time all your Backbone application views, models, collections and routers:

Views

Rendering status, html element, associated model and/or collection, handled page events, events triggered

Models

Last sync status, attributes, id, cid, url, associated collection, events triggered, sync actions

Collections

Last sync status, models, url, events triggered, sync actions

Routers

Events triggered (include routes)

Extends the sidebar of the Elements panel to display the Backbone View of the inspected html element.
